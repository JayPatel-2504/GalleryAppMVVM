# GalleryAppMVVM

A simple **iOS Gallery Application** built with **MVVM architecture**.  
The app allows users to log in with **Google Sign-In**, view a list of images with **pagination**, and works offline using **Core Data** and **SDWebImage** caching. It also monitors network connectivity and shows online/offline status.

---

## Features

- **Google Sign-In** authentication
- Gallery with **UICollectionView**
- **Pagination** for fetching images
- **Offline support** using Core Data + SDWebImage cache
- **Network monitoring** (Online / Offline label)
- **Global alert helper** for showing errors
- MVVM architecture for clean code

---

## Requirements

- Xcode 14.2 or higher  
- iOS 13.0 or higher  
- Swift 5.8 or higher  
- Swift Package Manager for 'SDWebImage'  and 'Google-SignIn'
